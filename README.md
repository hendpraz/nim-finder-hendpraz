## Cara Memasang dan Menjalankan Aplikasi

### Requirement

An adequate version of:
- nodejs & npm

### Cara Memasang dan Menjalankan Aplikasi

Langkah-langkah memasang dan menjalankan aplikasi:
1. cd ke lokasi repository
2. Jalankan `npm install` pada cmd
3. Jalankan `npm run dev` pada cmd.
